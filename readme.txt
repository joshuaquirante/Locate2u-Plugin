Connect your Locate2u app using this plugin without coding needed.

== Description ==

Locate2u Delivery Booking is a plugin where the client can integrate the Locate2u App without major programming needs.

Major features in Locate2u Delivery Booking include:

* Automatically integrate your website to the locate2u app.

== Installation ==

1. Download and Install Locate2u Delivery Booking Plugin in your wordpress site.
2. Active the plugin.
3. Enter your Locate2u ClientID and Client Secret in the Plugin Settings.
4. Create your page and select the template you want to choose.
5. You can choose either Single Stop or Shipment template.
6. ALL DONE!
